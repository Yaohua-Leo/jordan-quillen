# Experiment 005: Quillen Derived Indecomposables Toy Benchmark

Status: run

Plan: `researchplan/subplan005.md`

Goal: benchmark the first linear check for Quillen-style derived
indecomposables on one-generator Jordan algebra presentations.

This experiment records expected values for:

```text
F(x)
F(x)/(x^2 - x)
F(x)/(x^2)
F(x)/(x^3)
F(x)/(x^4)
```

Working conventions:

- Base field: `Q`.
- Jordan algebras are nonunital unless a file states otherwise.
- One-generator special Jordan algebras are treated as power-associative.
- After applying `Q`, only the linear part of a relation `r(x)` survives.
- This is a toy benchmark, not a full cofibrant replacement engine.

Run command:

```text
python experiments/005-quillen-derived-indecomposables/run.py
```

Result:

- `results.json` was generated by the run command.
- The five planned examples all match `expected.json`.
- The overall result is `passed: true`.

Boundary: this remains a toy derived-indecomposables benchmark. It is not a
proof of the full Jordan Quillen homology theory and does not claim
presentation invariance.
