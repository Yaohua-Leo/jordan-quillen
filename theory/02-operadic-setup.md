# 02. Operadic Setup

Working premise: over a characteristic-zero field, Jordan algebras are algebras
over the Jordan operad `Jord`.

The operadic route is useful because it gives a standard homotopical language:
dg Jordan algebras can be treated as `Alg_Jord(Ch_k)`, and Quillen homology can
be expressed as derived indecomposables.

## Proof Obligations

- Verify admissibility of the Jordan operad in the chosen model category.
- Decide whether the project uses simplicial Jordan algebras, dg Jordan
  algebras, or both.
- Record comparison functors if both settings are used.
