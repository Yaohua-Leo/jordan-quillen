# 11. Examples

Initial example list:

- Zero multiplication Jordan algebra.
- One-dimensional unital Jordan algebra.
- Square-zero extensions.
- Two-dimensional Jordan algebras.
- Special Jordan algebra `A^+` attached to an associative algebra `A`.
- Small examples with computable TKK Lie algebras.

Each example should record:

- base field and characteristic;
- multiplication table;
- module choice;
- derivations;
- expected `Omega`;
- whether the example tests a claim file.
