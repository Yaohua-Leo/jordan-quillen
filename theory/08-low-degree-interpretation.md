# 08. Low-Degree Interpretation

Expected low-degree meanings:

- `D^0_Jord(J/A, M)` should be `A`-linear Jordan derivations `J -> M`.
- `D^1_Jord(J/A, M)` should classify suitable square-zero extensions.
- `D_0^Jord(J/A)` should recover indecomposables in the absolute case.

These interpretations should be proved only after Beck modules and
`Omega_{J/A}` are fixed.
