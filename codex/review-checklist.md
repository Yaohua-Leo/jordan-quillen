# Review Checklist

- Are claims clearly marked as proved, draft, conjectural, or unread?
- Did any definition change require updating a claim file?
- Are local PDFs still ignored by git?
- Are experiment results real, or clearly marked `not run yet`?
- Do `python -m pytest` and `python scripts/check_claims.py` pass?
- If ruff is available, does `python -m ruff check .` pass?
